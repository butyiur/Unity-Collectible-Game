# Unity-Collectible-Game

Játék leírás:

Ez egy egyszerű, kezdő third-person Unity játék, ahol a játékos egy karaktert irányíthat WASD billentyűkkel. A pálya indulásakor különböző helyeken almák és coinok jelennek meg, amelyeket a játékos összegyűjthet.

Almák: minden összegyűjtött alma ehető, ez növeli a játkos életerejét. Az alap életerő 3/5, amely az alma elfogyasztásával tovább növelhető.

Coinok: a játékos pontszerzés céljából gyűjtheti őket.

Inventory rendszer: az I billentyű megnyomásával megnyitható az inventory panel, ahol a játékos valós időben láthatja, mennyi almát és mennyi coint gyűjtött össze.

A játék célja egyszerű: felfedezni a pályát, összegyűjteni a lehető legtöbb tárgyat, és minél tovább életben maradni, miközben a játékos visszajelzést kap az erőforrásairól egy letisztult inventory felületen.
